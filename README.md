# 👋 Olá, eu sou Felipe Neves

## 💻 Desenvolvedor Full Stack

Tenho 16 anos e sou estudante do 3º ano do curso Técnico em Desenvolvimento de Sistemas no CEEP Pedro Boaretto Neto.

Sou apaixonado por tecnologia, desenvolvimento de software e criação de soluções digitais. Busco constantemente aprimorar minhas habilidades em desenvolvimento web, arquitetura de sistemas e boas práticas de programação, sempre explorando novas tecnologias e desafios que contribuam para meu crescimento profissional.

Atualmente trabalho com tecnologias modernas para desenvolvimento Front-end e Back-end, desenvolvendo projetos que me permitem aplicar conhecimentos em interfaces, APIs, bancos de dados e versionamento de código.
